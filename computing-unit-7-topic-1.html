Integrated Development Environment
Revision
Quiz

An Integrated Development Environment (IDE) allows the user to write computer programs, using specialised tools to help them program. Examples of these tools are editors, debuggers and linkers.




Editor

An editor allows a programmer to enter, format and edit computer code. Different commands and functions may be different colours to allow a programmer to easily distinguish the different parts of the code. The editor is extremely important because without it, the user would not be able to enter the source code.




Compiler

A compiler converts the high level code into executable machine code for a later time (the entire program is converted, not one line at a time). It is slower to compile code than to interpret it but it can be executed faster once it has been compiled.




Interpreter

An interpreter converts the high level code one line at a time, into machine code and then executes it. It is easier to debug interpreted code as you can run each line at a time, searching for bugs. However, it is slower to execute than compiled code.
