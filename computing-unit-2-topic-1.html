Rules

A̅ = NOT A

A.B = A AND B

A+B = A OR B

A⊕B = A XOR B



Worked Example 1

Rules

  • P.1 = P

  • P.Q + P.R = P.(Q+R)

  • P+P̅ = 1
    • Expression

  • X = A.B + A.B̅

  • P.Q + P.R = P.(Q+R)

  • P+P̅ = 1


    • Step 1

  • Find the rule that looks most similar to the expression (Rule 2)
  • Use it to simplify the expression by substituting it into the rule

    • P.Q + P.R = P.(Q+R)

    P = A, Q = B, R = B̅

    X = A.(B+B̅)



    Step 2

  • Find the next closest rule (Rule 3)
  • Repeat the process

    • P+P̅ = 1

    P = B, P̅ = B̅

    X = A.1



    Step 3

  • Use the final rule to simplify it further

    • P.1 = P

    P = A

    X = A





    Worked Example 2

    Rules

  • P.P̅ = 0

  • (P+Q).R = (P.R) + (Q.R)

  • P+0 = P
    • Expression

  • X = (A+B).A̅

  • P.Q + P.R = P.(Q+R)

  • P+P̅ = 1


    • Step 1

    (P+Q).R = (P.R) + (Q.R)

    P = A, Q = B, R = A̅

    X = (A.A̅) + (B.A̅)



    Step 2

    P.P̅ = 0

    P = A, P̅ = A̅

    X = 0 + (B.A̅)



    Step 3

    P+0 = P

    P = B.A̅

    X = B.A̅
