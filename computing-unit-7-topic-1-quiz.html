Quiz
What program which allows previously compiled code to be linked together?

Linker

Loader

Compiler

Memory Inspector

The correct answer is A (Linker)

What is the function of a debugger?

To protect the program against malware.

To hide errors in the program.

To locate, identify and rectify errors in the program.

To cause glitches in the software

The correct answer is C (To locate, identify and rectify errors in the program)

True or False - It is faster to compile code than to interpret it.

True

False

The correct answer is False (It is slower to compile code than to interpret it)

Why is it easier to debug interpreted code?

Because it is executed faster

Because you can run each line at a time, searching for errors.

Because it is converted into machine code

Because the entire program is converted at once

The correct answer is B (Because you can run each line at a time, searching for errors)

A memory inspector displays the contents of what?

Cache Memory

The RAM

The CPU memory

A section of memory

The correct answer is D (A section of memory)

A breakpoint interrupts the program at a specific line of code, for what purpose?

To freeze the program

To restart the program

To alert the programmer when a variable has reached a specific value

To compare variable values against expected values

The correct answer is D (To compare variable values against expected values)
