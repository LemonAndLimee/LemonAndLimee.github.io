Advantages of a Network

  • Allows hardware, software and files to be easily shared
  • Allows for easy internal communication with other computers on the network. Can send emails to systems on and off the network
  • Data from computers on the network can be centrally backed up
  • It's easier to monitor computer activity if it is on a network
  • Users can log on to any computer on the network and access their files and data




    • Disadvantages of a Network

  • There is a cost to buy and maintain the server
  • Larger organisations will need to hire an administrator or network manager
  • If the server fails, all the network computers are affected
  • Security breaches and malware are morely likely
