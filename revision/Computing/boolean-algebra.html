Boolean Algebra
Notation


Ā = NOT A A.B = A AND B A+B = A OR B A⊕B = A XOR B





Worked Example


Rules

P.1 = P P.Q + P.R = P.(Q+R) P+NOT P = 1
Expression

X = A.B + A.NOT B




Step 1

  • Find the rule that looks most similar to the expression (Rule 2)
  • Use it to simplify the expression by substituting it into the rule

    • P.Q + P.R = P.(Q+R)

    P = A, Q = B, R = NOT B

    X = A.(B+NOT B)



    Step 2

  • Find the next closest rule (Rule 3)
  • Repeat the process

    • P+NOT P = 1

    P = B, NOT P = NOT B

    X = A.1



    Step 3

  • Use the final rule to simplify it further

    • P.1 = P

    P = A

    X = A







    Boolean Identities/Laws



    Commutative Law

  • This law switches around the order of expressions

    • A.B = B.A

    A+B = B+A



    Idempotent Law

  • This law eliminates repetition

    • A.A = A

    A+A = A



    Inverse Law

  • A AND Ā must be 0
  • A OR Ā must be 1

    • A.Ā = 0

    A+Ā = 1



    Identity Law

  • These expressions can be simplified to A

    • A.1 = A

    A+0 = A



    Annulment Law

  • A AND 0 must be 0
  • A OR 1 must be 1

    • A.0 = 0

    A+1 = 1



    Absorption Law

  • If the first A is true then both values in the brackets are true
  • If the first A is false then both values are false.
  • Therefore this equation relies entirely on A and can be simplified as just A.

    • A.(A+B) = A

    A+(A.B) = A



    Associative Law

  • This law swaps a value inside the brackets with a value outside the brackets.

    • (A.B).C = A.(B.C)

    (A+B)+C = A+(B+C)



    Distributive Law

  • The value outside the bracket is multiplied by the values inside the bracket
  • This forms two new brackets, linked by an OR/AND

    • (A+B)+C = (A+C).(B+C)

    (A+B).C = (A.C)+(B.C)



    Double Negation Law

  • The double NOT reverse the value back to A

    • NOT(NOT A) = A
