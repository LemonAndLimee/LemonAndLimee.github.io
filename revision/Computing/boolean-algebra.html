Boolean Algebra
Notation


Ā = NOT A A.B = A AND B A+B = A OR B A⊕B = A XOR B





Worked Example


Rules

P.1 = P P.Q + P.R = P.(Q+R) P+NOT P = 1
Expression

X = A.B + A.NOT B




Step 1

  • Find the rule that looks most similar to the expression (Rule 2)
  • Use it to simplify the expression by substituting it into the rule

    • P.Q + P.R = P.(Q+R)

    P = A, Q = B, R = NOT B

    X = A.(B+NOT B)



    Step 2

  • Find the next closest rule (Rule 3)
  • Repeat the process

    • P+NOT P = 1

    P = B, NOT P = NOT B

    X = A.1



    Step 3

  • Use the final rule to simplify it further

    • P.1 = P

    P = A

    X = A
