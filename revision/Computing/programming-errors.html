Programming Errors
Syntax Error


A syntax error is a mistake in the rules or grammar of the program

This means the program cannot be converted into executable form.

E.G. 'primt' instead of 'print'





Logical Error


A logical error is a mistake in the program, so the program works but produces the wrong results.

E.G. 'Total = Price - VAT' instead of 'Total = Price + VAT'





Rounding Error


A rounding error is where the program rounds a real/float to a fixed number of decimal places

This means the number becomes less accurate.

E.G. 3.125 rounding to 3.13





Runtime Error


A runtime/execution error is where the program stops as a result of an operation during execution.

E.G. Attempting to read line 8 in a file with only 7 lines





Linking Error


A linking error is where the compiler can't find a sub procedure.

The programmer may have declared it incorrectly or may not have imported the library.





Truncation Error


A truncation error is where the program truncates a real/float to a fixed number of decimal places.

This means the number becomes less accurate.

E.G. 3.125 truncating to 3.12
