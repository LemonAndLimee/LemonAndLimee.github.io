High Level and Low Level Languages
High Level Languages


A high level programming language uses code written in a way that is similar to a natural human language, such as English.

Programmers prefer to use high level programming languages as they are easier to understand, learn and program.

High level programming languages also allow the use of powerful commands that perform quite complex tasks.

High level languages are used when the execution speed is not critical.

A high-level language must be translated into a low-level language before it can be run, as they cannot be executed directly by the CPU.





Low Level Languages


Assembly language is a low level language consisting of abbreviations and symbols

It is more simple than high level languages , but easier to understand than pure machine code.

It is converted to machine code using an assembler.

Low level languages have a faster execution speed than high level languages.





Machine Code


Machine language is made up entirely of binary or hexadecimal instructions.

It is rarely typed directly into the computer as it is very long, and is susceptible to typing errors.

It is most often translated to from high level languages.
